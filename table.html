Having fun with form

A form is a way to collect data from users. It usually contains inputs, select, textarea, and buttons.

Usually a form has action property: which determines where the data is to be sent to when submitted. The method property determines "how" they data will be sent. Two available methods are GET and POST

Inputs

The type of input usually affects its appearance in the browser. An input can be of type: text, password, date,number, email, checkbox, radio, etc
